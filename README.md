# simulator (mipsim)

## version 1.0

### how to use
Make sure you can use `g++` and `c++11`, then, compile the programs:
```
$ make
```
Use mipsim
```
$ ./mipsim test/test.s
```

Type commands to execute assembler.
```
How to use mipsim:
	s | step | \n	: run step by step
	a | all		: run all
	r | reg		: print register
	m | memo	: print memory
	exit		: exit program
```
